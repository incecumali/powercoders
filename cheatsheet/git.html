Git Commands

  • Git use three different space to coordinate files. Workspace, Index and Repository

  • Initialize Git : to initialize git press windows button and then write cmd to open terminal after opening terminal. We have to tell Git where the "root" of the repository. use "git init < directory >".Do this just once

  • Check Status : to check status after reaching current directory write git status you

    see list of trecked and untracked files. color of tracked file most probably will be green and number of untracked files will be probably red

  • TrackFiles : to track file write git add filenameNow our files are in Index space.

  • git diff : If you do some changes in current files you can see all that changes by typing git diff

  • Unstage Files to untrack files git rm --cached <file name >.

  • after adding changes in index we can see differences between workspace and repoitory by git diff -cached

  • Git Push :by command git push we send files to repository

  • Git Log wih git log we can see all pushes on repository

  • Rename file : " git mv oldfileName newFileNam", after renaming file we change file name in working space on computer

    to change the file name in repository gitHub we should git add and git commit and git push the renamed file. Then github replace old name with new name

  • Delete file : git rm fileName

  • git add "file name" add all files in present directory
