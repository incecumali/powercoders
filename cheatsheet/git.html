Documentation of GIT- 30.11.2018

Gite use three different space to coordinate files. Workspace, Index and Repository

Initialize Git : to initialize git press windoes button and then write cmd to open terminal after opening terminal, go to the directory that you want ti initialize and write git init

Check Status : to check status ater reaching current directory write git status you see list of trecked and untracked files.color of tracked file most probably will be green and number of untracked file will be probably red

TrackFiles : to track file write git add filenameNow our files are in Index space. If you do some changes in current files you can see all that changes by typing git diff to see changes between workspace and repository you should write git diff -cached
