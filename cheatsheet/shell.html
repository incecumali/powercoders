cheatsheet of Topic 2/ SHELL

commands that Want to explain

  • cd

    • to detect current directory typ cd in terminal

  • dir

    • to list the items in directory typ dir in terminal

  • cd directoryname

    • to reach a special child directory of current directory typ cd diractoryname in terminal

  • dir ..

    • to turn back to the parent directory type cd .. in terminal

  • mkdir directory

    • to create a child directory typ mkdir directoryname in terminal

  • rmdir directory

    • to remove a child directory typ rmdir directoryname in terminal
